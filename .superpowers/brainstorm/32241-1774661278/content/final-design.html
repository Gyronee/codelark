Design Rules Summary

Batch Grouping
Consecutive same-tool calls merged.
Show count + file name list below.
Agent Nesting
Max 2 levels. Sub-agent indented.
Deeper agents flattened into parent.
Status Icons
⏳ running, ✓ done, ✗ error.
No animation, keep it simple.
Collapse Strategy
Agent done → collapse to 1-line summary.
Main task tools → truncate oldest (keep 5).
